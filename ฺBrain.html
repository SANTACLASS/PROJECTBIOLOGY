สมองคืออะไร? What is brain?

สมอง คืออวัยวะสำคัญในสัตว์หลายชนิดตามลักษณะทางกายวิภาค
จัดว่าเป็นส่วนกลางของระบบประสาท

1. สมองส่วนหน้า ( Forebrain )

1.ออลเฟกทอรีบัลบ์ ( Olfactory Bulb ) หน้าที่หลักในการดมกลิ่น สมองส่วนนี้ไม่มีการพัฒนามากนักในมนุษย์
2.ซีรีบรัม ( Cerebrum ) นี่คือส่วนสำคัญที่เกี่ยวกับการเรียนรู้และการเพิ่มพูนความสามารถต่างๆ เป็นศูนย์กลางการทำงานด้านประสาทสัมผัสของร่างกาย ส่วนนี้ยังแยกย่อยได้อีก 4 ส่วนคือ
3.ทาลามัส ( Thalamus ) เป็นศูนย์รวมของกระแสประสาทที่ผ่านเข้าออกทั้งหมด
4.ไฮโปทาลามัส ( Hypothalamus ) ศูนย์กลางของระบบประสาทอัตโนวัติ สร้างฮอร์โมนเพื่อควบคุมการผลิตฮอร์โมนจากต่อมใต้สมองอีกที การปรับอุณหภูมิของร่ายกาย อารมณ์ ความรู้สึก

2. สมองส่วนกลาง ( Midbrain )

ส่วนนี้ต่อเนื่องมาจากสมองส่วนหน้า และมี ออพติกโลบ ( Optic Lobe ) เป็นส่วนประกอบที่สำคัญที่สุด ทำหน้าที่ควบคุมการเคลื่อนไหวของนัยย์ตาและการเปิดปิดของรูม่านตาด้วย

3. สมองส่วนท้าย ( Hindbrain )

ส่วนท้ายสุดของสมองมีองค์ประกอบสำคัญอยู่ 3 ส่วนดังนี้
1.พอนส์ ( Pons ) ควบคุมการเคลื่อนไหวของกล้ามเนื้อใบหน้า เช่น การเคี้ยว การหลั่งน้ำลาย เป็นต้น
2.เมดัลลา ( Medulla ) ทางผ่านของกระแสประสาทจากสมองสู่ไขสันหลัง
3.ซีรีเบลลัม ( Cerebellum ) ตรงนี้คือส่วนที่เราเรียกกันว่า “ สมองน้อย ” เป็นส่วนหลักในการประมวลการรับรู้และควบคุมการสั่งงานของสมอง
